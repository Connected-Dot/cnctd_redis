# cnctd_redis

Redis wrapper with connection pooling and type-safe JSON operations.

## Installation

```toml
[dependencies]
cnctd_redis = "0.1.9"
```

## Features

- Connection pooling via r2d2
- Type-safe get/set with automatic JSON serialization
- Pub/sub messaging
- Hash operations
